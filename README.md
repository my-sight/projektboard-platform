# Projektboard Platform

Next.js + Supabase + Material UI  
Self-Hosted Kanban/Multi-Projektlösung.

**Setup:**
1. GitHub Codespace oder Vercel nutzen (kein Terminal nötig)
2. `.env.local` mit Supabase-Keys und Stripe-Daten anlegen
3. Loslegen!

**Technologien:**  
- Next.js (React)
- Supabase (DB/Auth)
- Material UI
- Stripe (Zahlung)
